# Planes de Implementación

Esta carpeta contiene los planes de features complejas.

## Estados
- `sin_implementar` — pendiente
- `en_progreso` — en desarrollo activo
- `implementado` — completado (con tag/commit de referencia)
- `cancelado` — descartado

## Formato

```markdown
# Plan: [Nombre]
**Estado**: sin_implementar
**Creado**: YYYY-MM-DD
**Implementado**: (commit hash o tag cuando se complete)

## Resumen
Qué problema resuelve.

## Tareas
- [ ] Tarea pendiente
- [x] Tarea completada

## Archivos a modificar
- `src/archivo.tsx` — descripción
```
